# XUtil-Sub

附加功能的Android工具类库

## APIs

* ### NotificationUtils -> 通知栏工具类

```
buildSimple                 : 构建简单的通知
buildBigPic                 : 构建带图片的通知
buildBigText                : 构建多文本通知
buildMailBox                : 构建带多条消息合并的消息盒通知
buildProgress               : 构建[带/无精确]进度条的通知
buildCustomView             : 构建自定义通知

notify                      : 通知
cancel                      : 取消通知
cancelAll                   : 取消所有通知
```
